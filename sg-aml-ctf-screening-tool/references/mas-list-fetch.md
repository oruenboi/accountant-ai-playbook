# MAS Targeted Financial Sanctions – How to Get the Lists

Primary source (latest):  
`https://www.mas.gov.sg/regulation/anti-money-laundering/targeted-financial-sanctions/lists-of-designated-individuals-and-entities`

If the page shows “service unavailable”, try:
- Retry in a normal browser (non-headless) or another network.
- Use the MAS mailing list subscription (on the same page) to receive updates by email.
- Search for direct file links (common patterns):  
  - `.../List-of-Designated-Individuals-and-Entities.xlsx` (TSOFA)  
  - `.../List-of-Designated-Individuals-and-Entities-UN-Act.xlsx`
- If all else fails, ask the user to paste the latest XLSX/PDF/HTML lists; confirm the “last updated” date.

Cross-check alternatives when MAS is unreachable:
- UN Security Council Consolidated List (authoritative upstream)  
  `https://www.un.org/securitycouncil/content/un-sc-consolidated-list`
- ISIL (Da’esh) & Al-Qaida List  
  `https://www.un.org/securitycouncil/sanctions/1267/aq_sanctions_list`
- Taliban 1988 List  
  `https://www.un.org/securitycouncil/sanctions/1988/materials`

Operational steps for the agent:
1) Ask the user for the MAS list file(s) or to confirm successful download from the URL above. Capture last-updated date.  
2) If they cannot access, supply the UN links above and request they paste results for the applicable regimes.  
3) Log which source was used (MAS direct, email, or UN fallback) and any gaps.
