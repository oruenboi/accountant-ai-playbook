# SG Corporate Secretarial (Private, Unlisted)

## Operating Rule
Draft **client-ready** documents by default. Auto-fill from provided records first. Ask only for true critical gaps.

## Quick Start
1. Load `references/compliance-checklists.md` for timeline/compliance checks.
2. For AGM drafting, load:
   - `references/field-mapping-agm.md`
   - `references/conditional-clause-rules.md`
3. Extract facts from latest BizProfile + FS before drafting.
4. Run quality gate (no avoidable placeholders, consistent names/dates/UEN).

## Core Workflows

### 1) AGM / AR Annual Cycle
1. Determine route: hold AGM or dispense AGM.
2. If holding AGM, prepare board resolution + AGM notice + proxy + minutes draft.
3. If dispensing AGM, prepare board minute + members' resolution + FS circulation note.
4. Prepare AR filing package and deadline tracker.

### 2) AGM Drafting Standard (Auto-Fill First)
1. Build field matrix using `references/field-mapping-agm.md`.
2. Fill every field available from source docs.
3. Apply conditional clauses via `references/conditional-clause-rules.md`.
4. If gaps remain, ask one compact question pack (time, venue confirm, chair, remuneration clause, re-election clause).
5. If defaults used, append `assets/assumptions-note.md`.

### 3) Beneficial Ownership & Nominee Lodgements
1. Maintain controller and nominee records + proofs.
2. Handle RORC updates within 2 business days of change.
3. Handle ROND/RONS updates within 2 business days of change.

### 4) Officers / Shares / Registered Particulars
1. Officer appointments/cessations/changes: prepare BizFile+ data within 14 days.
2. Allotments/transfers/reductions: prepare approvals, filing steps, and register updates.
3. Address/name/FYE changes: prepare resolutions and filing checklist within statutory windows.

## Inputs to Request
Request only missing essentials after intake:
- AGM time and venue override (if any)
- Chairperson override (if any)
- Whether remuneration and re-election resolutions should be included
- Any constitution-specific proxy or rotation rule

## Outputs to Deliver
- Client-ready AGM pack with minimal blanks (ideally signature/date only)
- BizFile+ filing action list with deadlines
- Assumptions note when defaults were applied
