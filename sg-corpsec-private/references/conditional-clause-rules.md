# AGM Conditional Clause Rules (SG Private, Unlisted)

Apply these rules to avoid generic/incorrect AGM wording.

## 1) Financial Statements Clause
Always include a clause to receive/adopt directors' statement + FS for the stated FYE.

## 2) Auditor Re-Appointment Clause
- Include only if the company has appointed auditor and AGM agenda requires re-appointment.
- Exclude for audit-exempt unaudited companies unless user explicitly requests inclusion.

## 3) Director Re-Election Clause
- Include only when constitution/practice requires retirement by rotation or user instructs inclusion.
- If unknown, ask user in the critical question pack.

## 4) Directors' Remuneration Clause
- Include only if there is directors' fee/remuneration to approve.
- If unknown, ask user in the critical question pack.

## 5) Proxy Clause
- Include proxy entitlement note and lodgement timing.
- Default deadline: not less than 48 hours before AGM unless constitution states otherwise.

## 6) Venue Defaults
- Default AGM venue to registered office if user gives no other venue.
- State this in assumptions note.

## 7) Chairperson Defaults
- Default chair to first listed current director if not specified.
- State this in assumptions note.

## 8) Minutes Outcome Defaults (Draft Mode)
- Use "carried unanimously" only for draft convenience and mark as assumption.
- For final signed minutes, replace with actual vote results/attendance.

## 9) Compliance Tone
Use formal SG corporate secretarial drafting style. Avoid conversational filler and vague placeholders.
