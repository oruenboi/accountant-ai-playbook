# Skillset.md QA Checklist

- All skills listed have matching folders or known names in scope.
- Each skill block includes: Description, Trigger phrases, Inputs, Outputs, Upstream dependencies, Downstream consumers, Key files, Notes.
- Dependencies only reference skills that exist in the inventory.
- Outputs from upstream skills map to inputs of downstream skills where applicable.
- Trigger phrases are user-facing and action-oriented.
- Conventions are filled (no TBD or placeholders).
- Orchestration constraints reflect actual dependency order.
- Formatting uses consistent headings and bullet styles.
